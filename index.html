EverCoin Bank
Commercial Fintech Web Mobile

Create a single platform (hub) to manage and promote the client's growing ecosystem of cryptocurrency tokens (Dragon Token, Cyber Token).

  1. EverCoin Bank has been fully developed as a working binding hub with up-to-date information, news, and graphs.
  2. The problem of complex listing has been solved by implementing an efficient process with Google Docs spreadsheets, which has significantly reduced the time for handling registration on exchanges.
  3. Traffic to ecosystem projects has been provided through EverCoin Bank.
  4. A scalable ecosystem has been created for the development of the ecosystem (adding tokens and trading).

03
ANT
Commercial PropTech Web App

The goal of the Ant project is to diversify the market by creating a competing platform. This should give users more choice and freedom, potentially reducing costs and creating a healthier competitive environment.

  1. In-depth design (Discovery) has been carried out: a roadmap (Backmen) has been defined, and the project architecture has been worked out.
  2. Complete analytical documentation has been prepared: detailed technical specifications and requirements for developers.
  3. Design work has been completed: UX/UX mockups of the entire client application and the admin panel have been created and approved.
  4. Development has entered the active phase: divided into four stages. The first stage, which includes the basic functionality of the admin panel, is currently being developed.

MVP versions have been developed for all platforms using the Scrum methodology, including the stages of analytics, design, and development. A subscription system with automatic billing has been implemented.
